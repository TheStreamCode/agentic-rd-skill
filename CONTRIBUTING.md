# Contributing

Contributions are welcome if they improve clarity, reliability, safety, or practical usefulness.

This project is maintained by Michael Gasperini. Website: `https://mikesoft.it`.

## Contribution Guidelines

- Keep prompts modular and reusable.
- Prefer specific instructions over vague advice.
- Avoid unsupported claims about agent autonomy or guaranteed outcomes.
- Include safety notes for regulated or high-risk domains.
- Use plain Markdown and clear English.
- Do not add generated outputs to the repository unless they are curated examples.

## Recommended Pull Request Scope

- One prompt improvement
- One guide improvement
- One template improvement
- One complete example
- One documentation clarification

## Style

- Use concise headings.
- Use imperative instructions for prompts.
- Use explicit acceptance criteria where possible.
- Avoid marketing language that overstates the capability of AI agents.

# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project follows semantic versioning where practical.

## [0.1.0] - 2026-05-09

### Added

- Initial prompt library structure.
- Master orchestrator prompt.
- Specialist agent, cross-review, stage-gate reviewer, and final synthesizer prompts.
- Markdown guides for workflow, agent selection, parallel research, review gates, final synthesis, and quality standards.
- Reusable templates for project briefs, specialist outputs, cross-review notes, stage-gate reviews, and final outputs.
- Example brief for an AI legal assistant SaaS.
- Documentation for safety, limitations, customization, and inspiration.
- Repository validation script and GitHub Actions workflow.
- Step-by-step guide for using the library with a coding agent.
- Self-contained `AGENTS.md` workflow entrypoint for single-file use in target projects.

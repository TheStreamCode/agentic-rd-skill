# Parallel Research Guide

Specialist agents should work independently before cross-review.

## Purpose

Parallel analysis reduces anchoring. Each specialist should form its own view before being influenced by other agents.

## Instructions For Specialists

- Read the project brief.
- Read the orchestration plan.
- Focus only on the assigned scope.
- Produce findings in the specialist output template.
- Declare assumptions and uncertainties.
- Ask questions for other agents when needed.

## Common Failure Modes

- Repeating the same generic analysis across agents
- Producing recommendations without evidence
- Ignoring constraints in the project brief
- Overstepping into another specialist's scope
- Hiding uncertainty

## Completion Criteria

Parallel research is complete when each selected specialist has produced a structured output with findings, evidence, risks, recommendations, and open questions.

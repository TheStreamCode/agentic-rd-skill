# Customization

This library is intentionally modular.

## Customize By Project Type

For business projects, emphasize market research, competitor analysis, business model, and go-to-market.

For technical projects, emphasize architecture, feasibility, security, implementation plan, and testing.

For scientific projects, emphasize literature review, methodology, experiment design, reproducibility, and limitations.

For problem-solving projects, emphasize root cause analysis, alternatives, decision criteria, implementation plan, and monitoring.

## Customize Agent Set

Add specialist prompts only when they improve the output. Avoid excessive agents when the brief is narrow.

## Customize Output Format

Use the final output template as a default. Replace it with a business plan, research report, product requirements document, or technical feasibility report when needed.
